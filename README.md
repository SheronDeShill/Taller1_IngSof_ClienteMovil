# TALLER 1 INGENIERIA DE SOFTWARE, SISTEMA DE CHAT (CLIENTE MOVIL)

Este sistema utiliza Dart/Flutter como lenguaje de programacion y framework

Este sistema NO requiere dependencias, para iniciar el sistema (de forma local) siga los siguientes pasos:

1.- Revisar los siguientes archivos: "chat_screen.dart", "auth_service.dart", "group_service.dart" y cambiar las URLs http://x.x.x.x:3000/ por la ip de la computadora en uso (consulte ipconfig en CMD) 
2.- conectar el computador con un celular a traves de un cable de datos
3.- permitir la transferencia de archivos en el celular
4.- activar las opciones de desarrollador -> depuracion por USB en el celular a utilizar
5.- iniciar el codigo (usando flutter run) en la carpeta cliente_movil y esperar (aprox 10m la primera vez). al terminar la aplicacion se iniciara automaticamente en el celular y dejara una aplicacion para abrise en cualquier momento.

NOTA: la base de datos solo funcionara mientras que el codigo este activo, al terminarse, la aplicacion no podra ejecutar más instrucciones, solo se podra acceder a las distintas interfaces que posee (modo offline).
